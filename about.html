About Golico!

Hello everybody. Golico is a term coined by Tauseef Hussain, developer of this static website hosted using Github pages. Golico is an amalgamation of the phrase - Goal linked co-operation/collaboration. As a developer, I wanted to showcase my skills in HTML5, JS, Bootstrap and usage of public APIs. Golico is an effort in that direction. It is inspired from a freely available Bootstrap theme - Stanley. A really simple theme for those wanting to showcase their work with a cute & clean style.

Stanley theme used for API development.

REST APIs used from Facebook, YouTube, GitHub.

Experimental setup for further development of other websites.

Facebook login mechanism explored.

PRIVACY POLICY

This website is hosted on Github pages for temporary duration till a suitable AWS hosting solution is found. It uses Facebook login to obtain certain information from the users who choose to grant access to their Facebook profile information. This website will not use information from the users for any marketing purposes. This website will not sell the information from the users to any third party companies or persons or organisations. The information collected from the users is purely for self consumption of the developers of the website.
